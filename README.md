Disaster Mesh Network is an offline-first communication system designed for situations where traditional networks fail (like earthquakes, floods, or disasters).
It enables device-to-device messaging using Bluetooth, WiFi Direct, and LoRa, allowing messages to travel across nearby devices in a mesh network — without requiring internet connectivity.
🎯 Problem
Network failure during disasters
No communication between people
Delayed rescue operations
Increased risk to lives
💡 Solution
A decentralized mesh-based communication system that:
Works without internet
Connects devices directly
Enables message relay across multiple nodes
⚙️ How It Works
Devices connect using Bluetooth, WiFi Direct, and LoRa
Each device acts as a node in the network
Messages are passed (hopped) from one device to another
Data eventually reaches the intended receiver

In a disaster scenario:
A user is stuck and sends a Signal Flare
Nearby devices receive the alert
Message spreads across the mesh network
Rescue teams or other users get notified
Faster coordination and response
