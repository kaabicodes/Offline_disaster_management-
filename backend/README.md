# Backend

This folder contains a simple Express backend for the project, but it is now optional.

The frontend mesh UI works offline without the backend, so you only need this server if you want to run the legacy API example.

## Run backend

Install dependencies from the project root if needed, then run:

```bash
npm run backend
```

## Available routes

- `GET /api/health` - health check
- `GET /api/data` - example API response
